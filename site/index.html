The rules of this site

WebCaretakers is a small, independent calculator hub run by Andrew Laws. The whole site is built on a short list of self-imposed rules:

  • No sign-up, ever. If a tool needs an account, something has gone wrong in the design.
  • Everything runs in your browser. The numbers you type never leave your device. There is no server-side maths, no logging of inputs.
  • Show the workings. Every calculator has a "Prove it" panel that explains how the number was arrived at. You should never have to take the result on faith.
  • Plain English. If a tool cannot explain itself without jargon, it has failed.

A quick, necessary word
